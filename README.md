# setting of the project :
1. First download the zip file of this project.
2. extract the folder and copy to the xampp/htdocs/...
3. open the xammp server and run apache and mySql.
4. open vs code and open the folder that you have pasted in htdocs
5. edit your database name.
6. open your browser and run the locahost/your project path and any file/.
7. For payment you have to make own khalti merchant account and replace with your private key.

#About
This is the mini project based on php. This project represent a demo version of e-commerce site. 
In this project there is strong backend logic including the authentication handling, password hashing, user and admin role authentication.
A payment method "khalti" has been included.
